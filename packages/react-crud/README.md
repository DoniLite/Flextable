[![npm downloads](https://img.shields.io/npm/dm/@flextable/react-crud.svg)](https://www.npmjs.com/package/@flextable/react-crud)

[![License: Apache-2.0](https://img.shields.io/npm/l/@flextable/react-crud.svg)](https://github.com/DoniLite/Flextable/blob/main/LICENSE)

Optional CRUD-dialog glue for
[`@flextable/react`](https://www.npmjs.com/package/@flextable/react): `useEntityEditor` and
`useStoreAsyncOperations` hooks for create/update/delete flows, decoupled from any specific backend
or toast library via injected adapters — bring your own `notify` callback and
`ErrorCodeAdapter` (a NestJS-shaped default is included as an opt-in preset).

## Install

```sh
bun add @flextable/react-crud
```
